# GitHub Profile QR Code Generator

A Python project that generates a QR code for my GitHub profile.  
Scanning the QR code directly opens my GitHub account.

## Features
- Generates QR code for GitHub profile  
- Simple and beginner-friendly Python program

- ## Technologies Used
- Python  
- qrcode (Python library for generating QR codes)
  
## Purpose
This project was created to practice Python programming and improve my GitHub portfolio.
